- Information: Images&Sequences

  NGS产生了Big and HD data

#### 算法和模型的区别
- 模型是对数据和现实情况的抽象，通常是基于数据的，由数学/统计方法构建。
- 算法是针对特定计算任务的明确求解步骤，不依赖于数据，偏重于计算层面获得解的方法。
